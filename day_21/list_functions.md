### len()

### append()

* a_list = [1, 2, 3]
  a_list.append(4)
  print(a_list) => [1, 2, 3, 4]
  
### extend()

* a_list = [1, 2, 3]
  b_list = [4, 5, 6]
  a_list.extend(b_list)
  print(a_list) => [1, 2, 3, 4, 5, 6]
